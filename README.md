# ASTRAL KPI Sandbox

An interactive teaching tool for ASTRAL's detection KPIs — learn the seven metrics by playing.

**Live demo:** https://YOUR-GITHUB-USERNAME.github.io/astral-kpi-sandbox/

ASTRAL reads a sea of AI-interaction logs, scores each session for biological-misuse risk, and
surfaces the few worth a human review. This sandbox lets someone with no statistics background
*feel* what the detection metrics mean: one shared synthetic world driven by a few dials, where
moving any dial makes every metric react at once.

Covers: F2 (North-Star) · Precision · False Positive Rate · Recall · ROC-AUC · NDCG@k · Attribution (weighted κ),
plus a live **north-star** better/worse signal, a Batch vs Operational mode, dark/light themes, and an
"Under the hood — for the curious" panel on every tab.

## Use it

It's a single, self-contained HTML file — no build, no server, no dependencies (fonts embedded,
zero external requests). Just open `index.html` in any browser, or visit the live demo above.
